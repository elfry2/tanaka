# tanaka
A PowerShell script to (almost) effortlessly install common web development tools, to support common workflows, using common techniques. Written with those who prefer to easily collaborate with others in mind. Powered by WinGet.

The script installs:
- Git
- GitHub CLI
- XAMPP 8.2
- Composer
- VSCode
- NodeJS LTS
- MongoDB LTS
- MongoDB Compass

## Installation
On PowerShell, execute (can be pasted at once)
```powershell
git clone https://github.com/elfry2/tanaka
cd tanaka
./Install.ps1
```
